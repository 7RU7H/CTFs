Produk ini adalah Perangkat Lunak yang dilisensikan kepada Anda oleh Belkin dan, apabila berlaku, oleh pemasok Belkin. "Perangkat Lunak" berarti setiap dan semua program firmware dan file terkait yang disediakan sehubungan dengan Produk ini; setiap dan semua program perangkat lunak, aplikasi atau "apl", dan file terkait yang disediakan sehubungan dengan Produk; semua versi modifikasi atau perbaikan untuk program tersebut (seperti yang disediakan melalui pembaruan berbasis web), semua versi lanjutan dari program tersebut, dan semua salinan program dan file tersebut. Perangkat lunak tidak meliputi Perangkat Lunak Sumber Terbuka (seperti yang ditentukan di bawah ini).


Yang kami maksud dengan "Anda," adalah pembeli, penerima, atau pengguna akhir lain Produk yang berisi Perangkat Lunak atau pembeli, penerima atau pengguna akhir lain Perangkat Lunak ini secara mandiri. "Anda" juga berarti seseorang yang telah mengunduh Perangkat Lunak ini dari situs web resmi, misalnya http://www.belkin.com atau dari pasar atau bursa aplikasi resmi, misalnya Apple App Store atau Google Play (masing-masing pasar atau bursa aplikasi itu dalam Perjanjian ini disebut sebagai "Bursa Aplikasi" dan secara kolektif sebagai "Bursa Aplikasi").
